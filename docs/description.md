# General description of the system
A card game where players compete to be the first to earn 30 coins by developing their farm

## Available Modes:
* Player versus Player (PvP) - two players compete against each other
* Player versus Computer (PvE) - the player competes against an AI opponent

## Core System Functions:
* Control of turn sequence for both player and AI

* Processing of game events (planting crops, placing animals, collecting income)

* Support for coin mechanics and field limitations to balance gameplay

* Management of card deck and player's hand size

* Victory and defeat condition checking
